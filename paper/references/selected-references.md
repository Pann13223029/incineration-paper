# Selected References For The Integrated Paper

This is the working reference set for the current one-paper strategy. It is not
the full thesis bibliography. It is the smaller set that directly supports the
paper's framing, methods, and interpretation.

## Use Rule

If a source is not serving the current article's framing, methods, or
interpretation directly, keep it out of the main paper and leave it in the
thesis-only bibliography.
